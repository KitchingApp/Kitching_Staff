### ✅기능 소개
#### 🔪프랩
<img src="https://github.com/user-attachments/assets/d5aa7b8b-87b0-44be-8731-703a7db40401" width="200px" />

- 프랩 확인/추가/완료 기능
- 당일 해야할 프랩을 관리하는 화면

#### 🍳레시피
<img src="https://github.com/user-attachments/assets/47f02549-6454-4bae-b9bc-1888ae73f2f1" width="200px" />

- 레시피 검색/확인 기능
- 레스토랑의 모든 레시피를 볼수있는 화면

#### 🗓️스케줄
<img src="https://github.com/user-attachments/assets/bc59f216-c4ee-4fc4-bb7d-d38d9b918d7c" width="200px" />

- 스케쥴 확인/신청 기능
- 스케줄 거절 시 거절사유 FCM 알림수신 기능
- 근무하는 인원의 스케쥴을 확인할 수 있는 화면

#### 📑발주목록
<img src="https://github.com/user-attachments/assets/b57a10c6-9bc5-43ea-a1ae-d10988c07464" width="200px" />

- 발주목록 확인 수량체크 후 복사하기 기능
- 발주해야할 품목을 한눈에 확인 후 발주할 수 있는 화면

#### 📌그 외

<img src="https://github.com/user-attachments/assets/f1f223ee-a044-4578-80cd-d2e5b6ad22e2" width="200px" /> <img src="https://github.com/user-attachments/assets/b7422efe-7b7d-4d36-b017-57178af48689" width="200px" /> <img src="https://github.com/user-attachments/assets/4cbf92a7-e867-4cae-8b94-4b454edc243a" width="200px" />

- 공지사항 확인, 댓글작성 기능
- 새로운 공지사항 등록시 FCM 알림 수신 기능
- 같은 레스토랑 근무 멤버 조회기능
